## Our Tools

### [WinWright](https://github.com/civyk-official/civyk-winwright) — Windows Automation for AI Agents

[![GitHub Release](https://img.shields.io/github/v/release/civyk-official/civyk-winwright?label=Release)](https://github.com/civyk-official/civyk-winwright/releases)
[![Stars](https://img.shields.io/github/stars/civyk-official/civyk-winwright)](https://github.com/civyk-official/civyk-winwright)
[![MCP](https://img.shields.io/badge/MCP-~59%20tools-0D9488)](https://modelcontextprotocol.io/)

Windows automation server for the [Model Context Protocol](https://modelcontextprotocol.io/). ~59 consolidated tools for desktop (WPF, WinForms, Win32), browser (Chrome/Edge via CDP), and system management.

- **Describe tests in plain English** — the AI agent discovers UI controls, performs actions, and records everything
- **Deterministic replay** — recorded scripts run without AI/LLM, no token costs, same results every time
- **Self-healing selectors** — automatically repairs broken UI references when layouts change
- **Zero-dependency** — self-contained binary (x64/ARM64), no .NET runtime needed

### [RepoIx](https://github.com/civyk-official/civyk-repoix) — Semantic Code Intelligence

[![PyPI](https://img.shields.io/pypi/v/civyk-repoix.svg)](https://pypi.org/project/civyk-repoix/)
[![MCP](https://img.shields.io/badge/MCP-14%20tools-0D9488)](https://modelcontextprotocol.io/)

Semantic code intelligence for AI coding agents via MCP. Your code never leaves your machine — 100% offline, no cloud, no telemetry.

- **14 consolidated MCP tools** — symbol search, call graphs, architecture analysis, quality checks, context packs
- **AI Context Cache** — persists code understanding across sessions, saving 80-90% tokens
- **Sub-2ms symbol search** — full index in ~3 seconds, powered by Tree-sitter and SQLite
- **12 languages** — Python, TypeScript, JavaScript, Java, Go, C#, Rust, Ruby, PHP, T-SQL, PL/SQL, Markdown
- **Multi-agent** — Claude Code, Cursor, Windsurf, GitHub Copilot, Gemini CLI, and more

### [Hatch](https://github.com/civyk-official/civyk-hatch) — Spec-Driven Development

[![PyPI](https://img.shields.io/pypi/v/civyk-hatch.svg)](https://pypi.org/project/civyk-hatch/)
[![MCP](https://img.shields.io/badge/MCP-Compatible-0D9488)](https://modelcontextprotocol.io/)

AI-powered spec-driven development CLI. Brings structure to AI-assisted software building — instead of ad-hoc prompting, it guides AI agents through a defined workflow.

- **Orchestrator** — single-command workflow: constitution → spec → plan → tasks → implement
- **Auto-resume** — survives AI context limits with zero progress loss
- **Reverse engineering** — analyze legacy codebases, assess tech debt, generate modernization plans
- **DeepWiki** — generate comprehensive AI-powered documentation from existing codebases
- **12+ AI agents** — Claude Code, GitHub Copilot, Cursor, Gemini CLI, Windsurf, and more
